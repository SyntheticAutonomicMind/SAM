# Test Workspace

This workspace is used for MCP tool testing.

## Contents

- `root_files/` - Files in root directory
- `subdir1/` - First level subdirectory
- `subdir2/nested/` - Nested subdirectory structure
- `data/` - Test data files

## Purpose

Validates that SAM's MCP tools can:
- Read files from any directory level
- Write files to root and subdirectories
- Search across directory structures
- Handle nested paths correctly
